# chatify-dev-haris

A real-time chat application built with **React**, **Node.js**,**Zustand** **Socket.io**, and **MongoDB**. This app allows users to register, login, and chat with others in real-time with online/offline status indicators.

## Features

- User registration and login with authentication
- Real-time messaging using Socket.io
- Online/offline user status
- Select and chat with specific users
- Responsive UI
- Conversation history saved in MongoDB

## Tech Stack

- **Frontend:** React, Zustand, React Icons, Tailwind CSS (or your CSS framework)
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Real-time Communication:** Socket.io

## Installation

1. Clone the repository:

```bash
git clone <my-repo-url>
cd Frontend
npm install
```
2. Install dependencies for the backend:
```bash
git clone <your-repo-url>
cd Backend
npm install
```
3. Create a .env file in the backend folder and add:
```bash
MONGO_URI=<your-mongodb-uri>
JWT_SECRET=<your-jwt-secret>
PORT=4000
```

Open http://localhost:3000 to see the app in action
